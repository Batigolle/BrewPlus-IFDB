Welcome to BrewPlus IFDB Etition Pages.

BrewPlus-IFDB (BrewPlus - Il Forum Della Birra Edition) is an Italian FREE brewing software and an open source beer recipe creation tool developed in Java and compatible with Linux, Mac and Windows. It is based on on Hobbybrew 2.0.3 and BrewPlus 1.5.0. It assists you to calculate color, bitterness, and other parameters for you while you add ingredients into the recipe. It includes BJCP 2015 style guidelines and a renewed tool for water and PH calculation.
